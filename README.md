# Introduction

- PHP Framework : Laravel 5.5^ 

# Requirement

- PHP requirement :  PHP 7.0+
- exif extension
- GD Library >=2.0 or Imagick PHP extension >=6.5.7
- MySQL 5.7 or higher

# Installation
```
composer update
```
```
php artisan migrate
```
```
php artisan db:seed --class=DbTableSeeder
```
```
npm install
```
```
npm run dev
```

Enjoy :flushed: :relieved: :satisfied:
